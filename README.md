# MindusLauncher
A launcher for the game Mindustry written with Java Swing.

## Compiling
Install JDK 8.

### Windows
Compile: `gradlew jar`\
Run: `gradlew run`

### *nix
Compile: `./gradlew jar`\
Run: `./gradlew run`
